# Waste-Classification
This project aims to develop an automated waste classification system for residential apartments using a Convolutional Neural Network (CNN). The goal is to solve the common problem of incorrect waste sorting in high-rise buildings by providing a reliable tool that distinguishes between three main streams: Recyclables, Organics, and Garbage. 


The project utilizes the Garbage Classification (12 classes) dataset from Kaggle. After manual cleaning and label mapping, we retained 14,570 images.
